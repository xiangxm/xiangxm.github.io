Android Studio在Windows和Mac下的常用快捷键

发表于 2016-10-26   |   分类于 Android Studio   |     |   阅读次数

Android Studio在Windows和Mac下的常用快捷键

查看变量或者方法的注释

Windows Mac
Ctrl + Q

打开Action

Windows Mac
java Ctrl + Shift + A Cmd + Shift +A

快速导入包，并且去除无效包import

Windows Mac
Ctrl + Alt + O

搜索某一个类

Windows Mac
Ctrl + N

搜索某一个文件

Windows Mac
Ctrl + Shift + N

全局搜索文件

Windows Mac
Double Shift Search everywhere

返回到上一次编辑的地方

Windows
Alt + Right/Left Go to next/previous editor tab

返回到上一个光标位置

Windows Mac
Ctrl + Alt + 左右方向键

跳转到某一行

Windows Mac
Ctrl + G

跳转到具体注释

Windows Mac
Ctrl + B , Ctrl + Click Go to declaration

跳转到上一个或者下一个方法

Windows Mac
Alt + Up/Down Go to previous/next method

跳转到上一个或者下一个错误、警告提示的地方

Windows Mac
F2 / F2 + Shift

跳转到父类方法

Windows Mac
Ctrl + U Go to super-method/super-class

成员列表视图

Windows Mac
Ctrl + F12 File structure popup

搜索并替换

Windows Mac
Ctrl + R Replace

搜索

Windows Mac
Ctrl + F Find

关闭单个TAB

Windows Mac
Ctrl + F4 Close active editor tab

删除单词

Windows Mac
Ctrl + Delete Delete to word end

快速显示提示

Windows Mac
Alt + Enter Show intention actions and quick-fixes

快速添加代码块

Windows Mac
Ctrl + Alt + T Surround with… (if..else, try..catch, for,

插入Getter和Setter方法

Windows Mac
Alt + Insert Generate code… (Getters, Setters, Constructors,

Override方法

Windows Mac
Ctrl + O Override methods

重命名文件，如果是在文件内部，则可以一次性重命名变量

Windows Mac
Shift + F6 Rename

插入事先定义好的模板

Windows Mac
Ctrl + J Insert Live Template

单步调试

Windows Mac
F8 Step over

跳出调试方法

Windows Mac
Shift + F8 Step out

单步调试进入方法

Windows Mac
F7 Step into

在TAB之间切换

Windows Mac
Ctrl + Tab Switch between tabs and tool window

格式化代码

Windows Mac
Ctrl + Alt + L

将当前方法代码块与前一个/后一个方法位置对调(光标需要移动到大括号位置)

Windows Mac
Ctrl + Shift + 方向上/下键

GSON数据格式化,生产Parcelable/Serializable数据。

对生命周期方法排序

在线请求json数据
