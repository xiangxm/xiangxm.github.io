Android Studio在Windows和Mac下的常用快捷键

Oct 27, 2016 | Android Studio | Hits

鉴于平时在Windows 和Mac环境下切换比较频繁，有些时候快捷键都混淆，所以在这里将比较常用的集中起来这样每次忘记了可以速查一下。一部分最频繁的比如删除某一行代码，复制一行代码等等都没有写下来，因为太频繁不会忘。

随着逐步的使用，也会不断Update上来:)

如有错误的地方，欢迎批评指正，Thanks very much:）

查看变量或者方法的注释

Windows Mac
Ctrl + Q Control + J

打开Action

Windows Mac
Ctrl + Shift + A Command + Shift +A

快速导入包，并且去除无效包import

Windows Mac
Ctrl + Alt + O Option + Enter

搜索某一个类(注意这里区分搜索一个file)

Windows Mac
Ctrl + N Command + O

搜索某一个文件（注意这里区分搜索一个classname）

Windows Mac
Ctrl + Shift + N Command + Shift + O

全局搜索文件

Windows Mac
Double Shift Search everywhere Double Shift

返回到上一次编辑的地方

Windows Mac
Alt + Right/Left Go to next/previous editor tab Command + Shift + back键

返回到上一个光标位置

Windows Mac
Ctrl + Alt + 左右方向键 Command + [ / ]

跳转到某一行

Windows Mac
Ctrl + G Command + L

跳转到具体定义

Windows Mac
Ctrl + B , Ctrl + Click Go to declaration Command + B ，Command + Click

跳转到上一个或者下一个方法

Windows Mac
Alt + Up/Down Go to previous/next method

跳转到上一个或者下一个错误、警告提示的地方

Windows Mac
F2 / F2 + Shift F2 / F2 + Shift

跳转到父类方法

Windows Mac
Ctrl + U Go to super-method/super-class Command + U

成员列表视图

Windows Mac
Ctrl + F12 File structure popup Command + F12

搜索并替换

Windows Mac
Ctrl + R Command + R

搜索

Windows Mac
Ctrl + F Find Command + F

关闭单个TAB

Windows Mac
Ctrl + F4 Close active editor tab Command + W

删除单词

Windows Mac
Ctrl + Delete Delete to word end Option + delete 删除前面内容

快速显示提示

Windows Mac
Alt + Enter Show intention actions and quick-fixes Option + Enter

快速添加代码块

Windows Mac
Ctrl + Alt + T Surround with… (if..else, try..catch, for,

插入Getter和Setter方法

Windows Mac
Alt + Insert Generate code… (Getters, Setters, Constructors, Command + N

Override方法

Windows Mac
Ctrl + O Override methods Command + N

重命名文件，如果是在文件内部，则可以一次性重命名变量

Windows Mac
Shift + F6 Rename Shift + F6

插入事先定义好的模板

Windows Mac
Ctrl + J Insert Live Template Command + J

单步调试

Windows Mac
F8 Step over F8 over

跳出调试方法

Windows Mac
Shift + F8 Step out Shift + F8 Step out

单步调试进入方法

Windows Mac
F7 Step into F7 Step into

在TAB之间切换

Windows Mac
Ctrl + Tab Switch between tabs and tool window Control + tab

格式化代码

Windows Mac
Ctrl + Alt + L Command + Option + L

将当前方法代码块与前一个/后一个方法位置对调(光标需要移动到大括号位置)

Windows Mac
Ctrl + Shift + 方向上/下键

大小写切换

Windows Mac
Ctrl + Shift + U Command + Shift + U

打开/折叠代码

Windows Mac
Ctrl + Plus/Minus Command + Plus/Minus

上下移动代码

Windows Mac
Alt + Shift + Up/Down Option + Shift + Up/Down

Windows Mac
